# Ifthenelse Tap

## How do I install these formulae?

`brew install ifthenelse/tap/<formula>`

Or `brew tap ifthenelse/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "ifthenelse/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
